# 产品说明：ChatTTS 本地批量生成与音色参考

## 高层目标
- 在本地稳定完成“文本 -> 语音”的批量生成
- 支持参考音频的零样本音色模拟（非 1:1 复刻）
- 提供可复现实验流程，方便定位“只读后半段 / 音色不像”的问题

## 主要用户
- 需要批量语音合成的中文用户
- 需要用参考音频做“相似音色”输出的内容生产者

## 主要用户情景
- 通过 CSV 批量生成多条语音，并合并成一条
- 传入参考音频 + 逐字转写文本，尝试相似音色输出
- 对比不同采样参数与参考音频清洗策略的效果

## 功能列表
- 读取 CSV（文本、说话人、参考音频、参考文本）
- 解析参考音频并提取音色签名
- 可选裁剪静音并归一化参考音频
- 生成单条语音与合并语音
- 输出日志并记录异常（空音频 / 近静音）

## 原子功能拆分
- CSV 读取与字段校验
- 文本清洗与数字替换
- 参考音频加载与裁剪
- 音色签名提取（spk_smp）
- 推理参数组装与生成
- 输出保存与合并

## MVP 范围
- 单机 CPU 可运行
- CSV 批量生成 + 合并输出
- 支持参考音频 + 参考文本的基本流程
- 能复现“音色不像 / 只读后半段”的现状并记录
